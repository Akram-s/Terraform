# Terraform Examples on AWS
Terraform is used to create, manage, update infrastructure resources such as physical machines, VMs, Security groups, containers and more.

This repo contains [Terraform](https://www.terraform.io/) code examples on AWS

## Quick Start

You must have an [AWS (Amazon Web Services)](http://aws.amazon.com/) account.

The code consists of Terraform examples using HashiCorp Configuration Language (HCL) on AWS (Amazon Web Services).

All the code is in the [projects](/projects) folder.

For instructions on running the code, please consult the README in each folder.

This is the list of examples:

* [01-hello-terraform](projects/01-hello-terraform) - Example of how to deploy a single server on AWS (Amazon Web Services) in free tier
